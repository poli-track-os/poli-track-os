# Support

## Best channel by request type

- Bug report: open a GitHub issue with the bug template.
- Feature idea: open a GitHub issue with the feature request template.
- Documentation problem: open a GitHub issue with the docs template.
- Security issue: follow [`SECURITY.md`](./SECURITY.md) and avoid public disclosure.

## Maintainer bandwidth

This is currently a small-maintainer project. Response time may vary, but clear,
well-scoped reports with reproduction steps are much easier to act on than broad
problem statements.

## What helps us help you

- Exact page or component affected.
- Screenshots for UI regressions.
- Browser and operating system for frontend issues.
- Proposed behavior if the current behavior is ambiguous.
